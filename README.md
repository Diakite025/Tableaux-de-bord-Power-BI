# Tableaux-de-bord-Power-BI

## Description
Ce projet consiste en la création de **tableaux de bord interactifs sous Power BI** afin de visualiser et analyser des indicateurs clés de performance (KPI).

Les dashboards permettent de faciliter la lecture des données et d’aider à la prise de décision.

## Outils
- Power BI  
- Données simulées ou issues d’un jeu de données pédagogique
